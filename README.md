# Altrodav Technologies — Data Analyst Task 1

## Overview
SQL analytics task covering student engagement, dropout risk,
and placement performance on a learning platform.

## Files
| File | Description |
|------|-------------|
| `Querys.sql` | 5 optimized PostgreSQL analytical queries |
| `INSIGHTS.md` | Business recommendations based on query results |
| `q1_avg_time_to_level50_by_college.csv` | Aptitude progression speed by college |
| `q2_high_failrate_tracks.csv` | High dropout risk learning tracks |
| `q3_students_with_7day_streak.csv` | Students with 7+ day engagement streaks |
| `q4_skill_completion_rates.csv` | Skill module completion rates |
| `q5_top10_colleges_by_placement.csv` | Top 10 colleges by placement rate |

## Key Findings
- High fail rates (42–55%) in advanced tracks need intervention
- Faster aptitude progression correlates with better placement rates
- 4,237 students maintained 7+ day streaks
- Completion drops sharply for advanced skill modules

## Tools Used
- PostgreSQL 15+
- SQL (Window Functions, CTEs, Aggregations)
